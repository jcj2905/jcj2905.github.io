Project Portfolio

Welcome to my Online Portfolio, the purpose of this website is to showcase a couple of projects that I have been working on for the past year. These projects aim to highlight the specific UX processes of Mobile and Web Development.

UX/UI Projects

  • Smart Glucose

    Smart Glucose is an Mobile Application which allows users to monitor their glucose levels with data visualization. Users can also input food for nutritional information. This application aims to educate diabetic users on their daily glucose intake.

    View Project

  • TAMU Wayfinding

    This mobile application makes it easier for students and visitors on campus to find their way around using the Texas A&M Bus Transportation System. This application hopes to improve travel time as well as decrease stress in finding new places on campus.

    View Project

  • Mashed News

    This is an ongoing project which collects multiple different news sources and pulls news articles from each source. The application then should perform Sentiment Analysis and TF-IDF to determine potential biases between sources.
